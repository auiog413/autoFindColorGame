# autoFindColorGame
A bot that play "Color Block" (Find different color). 
自动玩“寻找不一样的色彩”游戏脚本。

## Requirements:
- OpenCV 3.0
- Python 3
- pyscreenshot
- xdotool
- scrot

## How to use:

Just using `python3 findColor.py` to start the progress.
